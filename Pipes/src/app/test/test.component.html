Pipes can be used to filter or modify some data before displaying it.We can also create custom pipes and directives.

Here are some predefined pipes which are applicable to string type properties.

{{name | uppercase}}

{{name | lowercase}}

{{titleCaseWord | titlecase}}

{{name | slice:3:5}}

We also have the json pipe to represent the object in JS object notation.

{{person | json }}

There are also some predefined number pipes which can ne applied to numbers.

{{ 2.229 | number:'1.2-3'}}

{{ 45.343434 | number:'3.0-0'}}

{{ 45434.332222 | number:'4.5-7'}}

{{ 45434.332222 | number:'10.2-4'}}

We also have percent pipe which displays the number in %

{{ .29 | percent }}

There is a currency pipe to represent the number in currency. By default it represents the number in $. We can specify the currency code as param for custom currency.

{{ 45.3432 | currency }}

{{ 54.323 | currency:'GBP'}}

{{ 53454234.323 | currency:'INR'}}

We can also specify to show currency code instead of symbol.

{{ 53454234.323 | currency:'EUR' : 'code'}}

We also have the date pipe

{{ date }}

{{date | date }}

{{date | date:'short' }}

{{date | date:'shortDate' }}

{{date | date:'shortTime' }}

{{date | date:'long' }}

{{date | date:'longDate' }}

{{date | date:'longTime' }}

{{date | date:'medium' }}

{{date | date:'mediumDate' }}

{{date | date:'mediumTime' }}
